# College AI Agent

This project is an AI Agent designed to answer queries related to:

- Admissions
- Courses
- Fees & Scholarships
- Policies
- Important Dates

## Features

- Intelligent query classification
- Reasoning-based decision making
- Multi-source data retrieval

## Tech Stack

- LangChain
- FastAPI
- OpenAI GPT
- PostgreSQL / MySQL
